# MadHacks2025-ED_VH_CS_LA

A Java-based recipe management system for students with dietary restrictions and cooking constraints.

## Features

- Recipe submission and storage
- Filtering by allergens and cooking methods
- Web interface and command-line tools
- Docker containerization
